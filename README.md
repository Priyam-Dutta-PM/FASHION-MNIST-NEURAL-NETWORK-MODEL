# FASHION-MNIST-NEURAL-NETWORK-MODEL
This project implements a Convolutional Neural Network (CNN) to classify images from the Fashion-MNIST dataset using TensorFlow and Keras.

## 🔍 Project Highlights
- Data normalization
- CNN architecture (Conv2D + MaxPooling)
- Batch Normalization
- Dropout Regularization
- Early Stopping
- Learning Rate Scheduling
- Confusion Matrix
- Classification Report

## 📊 Dataset
Fashion-MNIST contains 60,000 training images and 10,000 test images of 10 clothing categories.

## 🚀 How to Run

Open the notebook in Google Colab and run all cells.

## 📈 Final Performance
Test Accuracy: ~93–94%
